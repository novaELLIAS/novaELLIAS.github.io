{% timenode 2018-06-08 v1.1.0 %}
结束了持续三个月的连续更新, 由 `v1.0.13 alpha` 更新至 `v1.1.0`.
{% folding, 重要的更新内容 %}
1. 启用 `valine` 评论系统.
2. 启用卜蒜子统计.
3. 启用字数统计.
4. 背景 导航栏 鼠标样式 footer等方面的美工优化
5. 使用 `WordPress` 托管全站全部静态图像
{% endfolding %}
{% endtimenode %}
